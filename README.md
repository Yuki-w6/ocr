## トラブルシューティング
- qt.qpa.plugin: Could not find the Qt platform plugin "offscreen" in ~

QT_QPA_PLATFORMを指定することで解決。以下を実行した後に再度実行
```bash
export QT_QPA_PLATFORM=xcb
```
